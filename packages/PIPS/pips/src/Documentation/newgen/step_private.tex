\domain{import entity from "ri.newgen"}

\domain{map_entity_int = entity->int}

\domain{step_clause = reduction:map_entity_int + private:entity* + shared:entity* + transformation:int + nowait:unit}

\domain{step_directive = type : int x persistent block : statement x clauses : step_clause*}

\domain{step_directives = persistent statement->step_directive}

\domain{map_effect_bool = persistent effect -> bool}

The propagation origines of the SEND (and RECV) region is tracked by the
 \texttt{step\_point} and \texttt{map\_step\_point} domains. For a SEND (or RECV) region,
the \texttt{step\_point} associated in the \texttt{map\_step\_point} table gives the
original SEND (or RECV) region before propagation (fields module, stmt and data).

\domain{step_point = module : entity x persistent stmt : statement x persistent data : effect}

\domain{map_step_point = persistent effect -> step_point}

\domain{step_comm = path: map_step_point x interlaced: map_effect_bool x partial: map_effect_bool}
