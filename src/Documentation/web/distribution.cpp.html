PIPS Distribution

The PIPS software is distributed both in an executable form (for SUNOS4 and for SOLARIS2) and source code. Note that the source code is not bullet-proof and is large (200,000 lines of C code). Its compilation requires many public-domain tools to be installed, thus we recommend that you download the compiled version if you do not plan to use PIPS as a base for developing new analyses or transformations phases.

When installing PIPS on your system, please read the README and INSTALL files at the root directory of the distribution for further instructions on how to run or compile PIPS. For help mail to pipsgroup@cri.ensmp.fr.

PIPS executables

PIPS executables are available for SUN workstations, under SunOS or Solaris.
  • PIPS for SunOS4
  • PIPS for SOLARIS2 (Ultra Sparc long long integers version, toolpack 1.2 and daVinci 2)

PIPS and tools at the source level

PIPS and public domain tools PIPS is built upon are available at the source level. Note that the code is distributed as is with no warranty, even implicit, of any kind. It is distributed under the term of the GNU general license that you may find quite easily on the net.

GNU make, cproto, flex, bison and some other tools are required to PIPS. See the next section for pointers on how to find and download these tools which are not included in the basic distribution.

  • PIPS sources (includes Newgen and Linear)
  • NEWGEN (software engineering tool)
  • LINEAR C3 library (integer linear programming modules)
  • XPOMP (X-based 2D data graphical display)

Other softwares of interest

Here are pointers to some external software that is used by PIPS or by PIPS generated codes or for building the PIPS software. This sofware is generally freely distributed.
  • PVM 3.3: for the runtime of HPFC and WP65.
  • GNU software (gmake gm4 gtar gzip gcc emacs readline malloc gsed).
  • Da Vinci graph displayer.
  • Cproto 4.3 (automatic prototype generation for C).
  • Rman 2.4 (conversion of man pages to html).
  • Fortran Split (split a Fortran file into one file per function).
  • PERL 5.
  • LaTeX2Html is also be helpful.
  • Toolpack (kit of Fortran to Fortran transformations structurations and so on, written in Fortran).
