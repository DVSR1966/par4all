PIPS Distribution

The PIPS software is distributed both in an executable form and source code. Note that the source code is not bullet-proof and is large (200,000 lines of C code). Its compilation requires many public-domain tools to be installed, thus we recommend that you download the compiled version if you do not plan to use PIPS as a base for developing new analysis or transformation phases.

When installing PIPS on your system, please read the README and INSTALL files at the root directory of the distribution for further instructions on how to run or compile PIPS. For help mail to pipsgroup@cri.ensmp.fr.

PIPS executables

PIPS executables are available for SUN workstations, under SunOS or Solaris, under IBM Aix and under Linux. The installation of binaries requires 10-20MB. No that Pips has been compiled successfully on a DEC alpha.
  • GET(PIPS for Sparc Solaris 2,pips_GNUSOL2LL.tar.gz) (Ultra Sparc long long int version) (3.3MB)
  • GET(PIPS for Sparc SunOS 4, pips_GNUSOL1.tar.gz) (3.5MB)
  • GET(PIPS for IBM AIX 4,pips_IBMAIX.tar.gz) (3.7MB) (does not include wpips and epips, but only basic intefaces).
  • GET(PIPS for i86 Linux,pips_LINUXI86LL.tar.gz) (I86 long long int version) (3MB)
  • GET(Public parts for Solaris 2,public_GNUSOL2LL.tar.gz) (toolpack 1.2 and daVinci 2, pips can be run without these, just download if you really need these functionalities) (caution! 10MB!)

PIPS and tools at the source level

PIPS and public domain tools PIPS is built upon are available at the source level. Note that the code is distributed as is with no warranty, even implicit, of any kind. It is distributed under the term of the GNU public license (GPL) that you may find quite easily on the net.

GNU make, cproto, flex, bison and some other tools are required to build PIPS. See the next section for pointers on how to find and download these tools which are not included in the basic distribution.

  • GET(PIPS sources,pips_sources.tar.gz) (includes Newgen, Linear and Pips) (3.9MB)
  • GET(PIPS validation,pips_valid.tar.gz) (for non regression tests, to be added to the sources) (1.2MB)
  • GET(NEWGEN sources,newgen_sources.tar.gz) (standalone software engineering tool) (0.3MB)
  • GET(LINEAR C3 library sources,linear_sources.tar.gz) (standalone integer linear programming modules) (0.5MB)
  • GET(XPOMP sources,xpomp_sources.tar.gz) (standalone X-based 2D data graphical display) (0.2MB)
