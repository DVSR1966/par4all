Introduction

 PIPS is an automatic parallelizer for scientific programs. It takes as input Fortran 77 codes. It emphasizes interprocedural techniques. The analyses are driven by a make system (pipsmake) and stored in a database (pipsdbm). The compiler is made of phases that are called on demand to perform the analyses or transformations required by the user. There is a shell interface (Pips) and a window interface (wpips) to the parallelizer.

PIPS is being developped at CRI since 1988. The initial design was made by Rémi TRIOLET, François IRIGOIN and Pierre JOUVELOT. PIPS is built on top of two tools. The first one is Newgen which manages data structures. It builds manipulation functions from a declaration file. It allows to deal with persistent data and a hierachie of types. All PIPS data-types are based on Newgen. The second tool is the Linear C3 library which handles linear formula and structures based on these such as polyhedrons.
