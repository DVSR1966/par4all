SAXPY (Single-precision real Alpha X Plus Y) combines scalar 
multiplication and vector addition. It is one of the functions
in BLAS (Basic Linear Algebra Subpackages) package.
(http://en.wikipedia.org/wiki/SAXPY). The one presented here 
is written in C 99.

See ../README.txt to get the generic commands to parallelize, build and execute 
this example. There is no specific command needed for this example.
